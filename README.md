## Was sind die Ziele dieses Projekts?

## Relevante URLs

## Wie kann ich das Projekt aufsetzen?

### Voraussetzungen

- Node v10
- npm

### Setup/Installation

```
npm install
```

### Wie kann ich das Projekt benutzen?

### Benutzung

```
npm start
```

### Deployment

```
npm run deploy
```
